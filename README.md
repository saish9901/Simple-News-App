# News App

A simple and responsive news app built with React.  
Users can browse and read the latest news with smooth and easy navigation.

## Tech Stack

- React
- Tailwind CSS
- Context API
- React Icons
- Axios
- DaisyUI

## Features

- Clean and simple UI
- Fast navigation between pages
- Real-time news fetching using APIs
- Responsive design
- Category-based news browsing

## Techniques Used

- Debouncing
- Context API for state management
- Clean folder structure

## Installation

```bash
git clone <your-repo-link>
cd news-app
npm install
npm run dev
```
